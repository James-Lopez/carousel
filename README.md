# carousel
A slideshow component for cycling through elements like a carousel.

## Useage
```bash
$ mongod
$ npm install
$ npm run seed
$ npm start
```

## Uses
* HTML5
* CSS3
* Modern JavaScript (ES2015+)
* Moment
* Moment Timezone
* MongoDB
* Express
* Node.js
